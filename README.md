# STATS_414_Final_Project

Code for 414 Final Project.

Contents include: 
- EDA.ipynb 
- preprocessing.py
- ctgan_model.py
- dpctgan_model.py
- great_model.py
- realtabformer_model.py
- utility_evals.py
- fidelity_evals.py
- privacy_evals.py

Datasets for EDA.ipynb and preprocessing.py can be found at https://www.kaggle.com/datasets/xiaojiu1414/digix-global-ai-challenge. 

Run preprocessing.py for dataset for ctgan_model.py, dpctgan_model.py, great_model.py, and realtabformer_model.py.

Run preprocessing.py for real dataset and ctgan_model.py, dpctgan_model.py, great_model.py, and realtabformer_model.py for synthetic datasets for utility_evals.py, fidelity_evals.py, privacy_evals.py.
